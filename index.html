Projects

Visitado

Inspired by my own desire to track the places I have been to, I developed and deployed a web application to track the places you have visited and write about your experience. I designed a RESTful API backend enabling easy interactions with the NoSQL database that stores all user data. I also implemented authentication with OAuth, integrated multiple external APIs to find locations, display interactive maps, search for thumbnail pictures The application was deployed to Digital Ocean using Docker-Compose and Nginx.

For this project I used: JavaScript, NodeJS, Express, MongoDB, Docker, HTML

Chat App

I created and deployed a chat web application that allows users to exchange messages in real-time. Users must first register and then login. Their password is stored safely in a PostgreSQL database. Users can enter and leave the various rooms. Every message is stored in and loaded from the database. Users can search and send Gifs obtained from the Giphy API.

For this project I used: Python, Flask, WebSockets, PostgreSQL, Docker, HTML
