I love working on personal projects as each one provides me an opportunity to learn, put my skills to the test and create cool applications. Below are some of the projects I have created in the past year. All of these projects can be viewed or played in your browser so be sure to check them out!

Stock Trading Simulator

Interested by the world of finance, I decided to create a full stack web application that allows users to learn about investing by buying/selling stocks risk-free. I created a dynamic frontend with React and a RESTful API backend using NodeJS and Express. I implemented authentication with JSON Web Tokens, data visualizations of stocks using Chart.js and a stock purchase and selling system. To get updated data about the market, I used several external APIs.

For this project I used: JavaScript, React (Hooks, Routing), NodeJS, Express, MongoDB, APIs
Check it out: oktarian-trading-simulator.herokuapp.com

Visitado

Inspired by my own desire to track the places I have been to, I developed and deployed a web application to track the places you have visited and write about your experience. I designed a RESTful API backend enabling easy interactions with the NoSQL database that stores all user data. I also implemented authentication with OAuth, integrated multiple external APIs to find locations, display interactive maps, search for thumbnail pictures The application was deployed to Digital Ocean using Docker-Compose and Nginx.

For this project I used: JavaScript, NodeJS, Express, MongoDB, APIs, Docker, HTML
Check it out: visitado.co.uk

Chat App

I created and deployed a chat web application that allows users to exchange messages in real-time. Users must first register and then login. Their password is stored safely in a PostgreSQL database. Users can enter and leave the various rooms. Every message is stored in and loaded from the database. Users can search and send Gifs obtained from the Giphy API.

For this project I used: Python, Flask, WebSockets, PostgreSQL, HTML
Check it out: chat-app-oktarian.herokuapp.com
